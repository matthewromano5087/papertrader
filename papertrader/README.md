# PaperTrader

## Quickstart

On Linux & Mac,

```
$ python3 -m venv myenv
$ source myenv/bin/activate
(myenv) $ pip install -r requirements.txt
(myenv) $ uvicorn papertrader.app:app
```

On Windows

```
> python3 -m venv myenv
> myenv\Scripts\activate.bat
(myenv) > pip install -r requirements.txt
(myenv) > uvicorn papertrader.app:app
```
